# Getting Started

Kajoot.it is is a simple website that displays quizzes from api https://opentdb.com/

### Tech Stack
- React
- Tailwindcss
- Vitejs
- Typescript
  

### How to use :
- git clone ```https://github.com/zaiinhs/kajoot.it.git```
- cd ```kajoot.it```
- npm install
- npm run dev
- open http://localhost:3000/ in local browser
